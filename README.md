# Mancala AI Agent

This project implements a Mancala-playing AI agent using adversarial search.

The agent uses the **Minimax algorithm with alpha-beta pruning** to evaluate game states and choose optimal moves.

## Features

- Complete Mancala game implementation
- Game state representation and board logic
- Adversarial search using minimax
- Alpha-beta pruning for optimization
- Heuristic evaluation of board states

## Technologies

- Python
- AIMA Python AI utilities

## Background

This project was developed for CSCI 3202 (Introduction to Artificial Intelligence) at the University of Colorado Boulder.

The goal was to implement a competitive AI agent capable of playing Mancala using classical AI search techniques.
